Instructions to run:
 - For Unity, Install Newtonsoft JSON package. Then create an empty gameobject in the hierarchy, Attach Level Parser and then create a child transform in it and plug it in the Level Parser Script. When loading levels, input the level file to 
Level File field and click Load Scene.

- For the scene editor, Level files are outputted and imported in the same directory as the solution
